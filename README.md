# Dr. Crow

An educational Telegram bot serving 400+ university students at 
Daffodil International University, Bangladesh.

## What it does
- Exam resource delivery and management
- Academic announcements and broadcast targeting
- Inline search for study materials
- Bot analytics and admin dashboard
- Slides and resource category system

## Tech Stack
- Python (aiogram / python-telegram-bot)
- Supabase (PostgreSQL)
- Telegram Bot API 10.1

## Architecture
~25,000 lines of Python across handlers, services, 
middleware, scheduler, and database layers.

## Community
Built for and maintained by the Twilight Crows community 
(~400 members, DIU Batch 66).
